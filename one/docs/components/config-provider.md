## API

### 属性

| 名称 | 类型 | 默认值 | 描述 |
| -- | -- | -- | -- |
| ``value`` | `Object` | - | 向其后代组件提供的配置值。 |

### 插槽

| 名称 | 描述 |
| -- | -- |
| ``default`` | 需要接收配置的内容，其中 VEUI 的组件会响应对应的配置值。 |
