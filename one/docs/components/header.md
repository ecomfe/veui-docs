# Header <small>布局头部</small>

:::tip
`Header` 组件需要配合 [`Layout`](./layout) / [`Sidebar`](./sidebar) / [`Footer`](./footer) / [`Content`](./content) 组件使用。
:::

## API

### 属性
| 名称 | 类型 | 默认值 | 描述 |
| -- | -- | -- | -- |
| ``sticky`` | `boolean` | `false` | 是否吸顶，使用注意参考 [`吸顶/吸底布局`](./layout#吸顶吸底布局)。 |

### 插槽

| 名称 | 描述 |
| -- | -- |
| ``default`` | 布局头部的内容。 |
